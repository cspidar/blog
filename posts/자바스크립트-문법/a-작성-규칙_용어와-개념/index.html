  • 반환값, 기존 대상의 변경 여부, 항목의 생략 가능 여부 (+생략했을때 기본값) 을 명확히 하는데 중점을 뒀습니다.

  • 학습 과정중, 정리를 위해 작성하여 오류가 있을 수 있습니다. 이에 대해 제 스스로 알게되거나, 관련 내용을 전달해 주시면 업데이트 하도록 하겠습니다.

  • 오류 수정 외에도 관련 예시와 활용법 추가는 지속적으로 진행할 예정입니다.

  • 본 문서의 첫번째 작성 목적은 제 학습에 대한 정리와 리마인드를 용이하게 하려는 것이나,
    자바스크립트 학습을 시작하시는 분들에게는 파악이 필요한 대상의 요약본이, 숙련자분들에게는 리마인드에 도움이 되는 도구가 되었으면 하는 바램입니다.

x1

시작 요소

…​x

x

특정 요소 (요소 리스트중 1개 지칭)

1.2. 인덱스#

start

시작 인덱스

end

끝 인덱스

target

특정 인덱스

차례로

start → end

역순으로

end → start

2. Array (배열)#

2.1. 작성 규칙#

Array

arr = []

index

i

value

v

2.2. 예시#

arr = [v1, v2, ...v]
arr[i1] = v1

3. M x N Matrix (M x N 행렬)#

3.1. 작성 규칙#

행

i (y 좌표)

열

j (x 좌표)

3.2. 예시#

matrix = [
[i1, j1], [i1, j2], [i1, ...j],
[i2, j1], [i2, j2], [i2, ...j],
[...i, j1], [...i, j2], [...i, ...j]
]

k

v

6. Map (맵)#

6.1. 작성 규칙#

Map

map = { => }

key

k

value

v

6.2. 예시#

map.set(k1, v1)
map.get(k1) = v1

8. Function (함수)#

8.1. 작성 규칙#

Function

func = (){}
func = () ⇒ {}

parameter

p

연산 함수

p ⇒ p + 1

판별 함수

p ⇒ p < 10

8.2. 예시#

func = (p1, p2, ...p) {p1, p2, ...p}
func = (p1, p2, ...p) => p1, p2, ...p
